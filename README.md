# Lazy-Iterator

The LazyIterator class as found in symfony/finder extracted to a composer package.

## Installation

### Prerequisites 

* php: `^7.4|^8.0.0|^8.1`

### Composer

```sh
composer require jascha030/lazy-iterator
```
